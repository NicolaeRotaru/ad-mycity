- [2026-07-17] **File vault NON vanno nel branch codice — causano conflitti inevitabili (PR #429 chiusa per questo, 17/7):** quando creo un branch per un fix, qualsiasi commit di memoria/vault su main locale contamina il branch e crea conflitti.
