- [2026-07-17] **Pattern git: prova SEMPRE da solo prima di chiedere i comandi a Nicola (PR #427, 17/7):** la disponibilità di git/Bash varia per sessione — non assumere che sia bloccato. - [2026-07-17] **Typecheck scaduto in sessione rapida — se main avanza dopo il typecheck, il verde è falso (Nicola: "stai creando errori su errori"):** in sessioni con molte PR ravvicinate, c'è un gap tra "typecheck verde" e "PR aperta". Se nel frattempo un'altra PR mergia sullo stesso file, il typecheck è obsoleto. Root di stanotte: typecheck verde su page.tsx → PR #423 mergia → git-pr.mjs rebase include #423 → possibile errore TS non catturato → build Vercel rosso (PR #424). - [2026-07-17] **Branch stale in sessione rapida = conflitti certi (Nicola: "non c'era fino a ieri"):** se main accumula molte PR sullo stesso file (page.tsx) mentre un fix-branch è fermo, anche di poche ore, la PR non si mergia.
