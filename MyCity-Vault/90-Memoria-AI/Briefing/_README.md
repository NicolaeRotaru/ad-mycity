# 📥 Briefing — i giri di perlustrazione dell'AD

Qui l'AD salva **un file per ogni giro** (`AAAA-MM-GG.md` o `AAAA-MM-GG-HHmm.md`).

Ogni briefing contiene:
- **Situazione** — cosa è emerso dai dati, in 2-4 frasi.
- **Opportunità** — dalla più grande alla più piccola (impatto × sforzo).
- **Azioni proposte** — concrete, ciascuna con colore 🟢🟡🔴.

Le decisioni che approvi vanno poi nel registro [[DECISIONI]], e i numeri nel
cruscotto [[STATO]].
